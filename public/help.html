AI Guard Manager Portal - Complete Help & Installation Guide

Server Installation

  • PHP 8.0 or higher
  • MySQL 5.7+ or MariaDB
  • Extensions: curl, yaml, mbstring, json
  • Composer: run composer install from root
  • Writable folders: /data/persona, /data/guardrails, /vendor
  • Recommended: SSL (HTTPS), Apache/Nginx with PHP-FPM

Database Setup

CREATE DATABASE ai_guard_manager;
GRANT ALL PRIVILEGES ON ai_guard_manager.* TO 'aiguard'@'localhost' IDENTIFIED BY 'yourpassword';

-- Tables include: users, app_settings, bug_tracker, file_audit_log
-- Use included SQL file or setup wizard

Environment Setup

  • Set OPENAI_API_KEY in your environment or php.ini
  • Update includes/db.php with DB credentials
  • Run install_checker.php to validate configuration

Core Modules

  • Dashboard: Central status page
  • Bug Tracker: Submit/edit bugs with priority and status
  • Admin Console: App settings, audit log, install check
  • Persona Manager: Define YAML personas
  • Guardrail Manager: Define policies, actions, logging
  • SSO & User Admin: User roles, audit, login control

Help Topics

Troubleshooting

  • 500 Errors: Check logs in /var/log/nginx/error.log or /var/log/php*
  • SSO Not Working: Confirm correct redirect URI and discovery document
  • Install Check Failures: Rerun install_checker.php and review [FAIL] items
  • Permissions: Ensure www-data owns all writable directories
